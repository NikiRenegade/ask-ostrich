# Best Strous-troupe team Site

## Требования

- Node.js (version 16 or higher)
- npm or yarn

## Запуск приложения

1. **Установить зависимости**:
   ```bash
   npm install
   ```

2. **Запустить приложение**:
   ```bash
   npm run dev
   ```

3. **Просмотр**:
   Открыть браузер по адресу `http://localhost:5173`
